# Pro Graph

A Graph Neural Network for predicting protein-protein interactions (PPI).

## Status

🚧 In progress.

## Approach

1. ⬜ Build the protein interaction graph from a benchmark dataset (Yeast PPI)
2. ⬜ Baseline model: Graph Convolutional Network (GCN) for link prediction
3. ⬜ Stronger model: Graph Attention Network (GAT)
4. ⬜ Optional: incorporate pretrained protein embeddings (ProtBERT) as node features
5. ⬜ Compare results, document findings
